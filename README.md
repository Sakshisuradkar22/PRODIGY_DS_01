# Task 1: Data Visualization

## Objective
Visualize World Bank population data using a bar chart and histogram.

## How to Run
1. Open `Task1_visualization.ipynb` in Jupyter Notebook or VS Code.
2. Make sure you have these libraries installed:
   - pandas
   - matplotlib
   - seaborn
3. Run all cells.

## Files
- `Task1_visualization.ipynb` → Your code
- `sample_dataset.csv` → Population data

## Knowledge Gained
- Loading CSV files with pandas
- Creating bar charts and histograms
- Basic data visualization in Python
